This is How to use section

Follow these step:

  1. Create a google sheet and let General access "Anyone with the link" with "Editor" role
  2. Edit sheet with this field name respectively: x, y and t (where x is liquid fraction, y is vapor fraction and t is temperature at Celcius)
  3. All sheet name will be showed up on composition list on the app e.g: ethanol-water, methylic-benzene, ... Make sure you sheet name is composition name
  4. After add url to your google sheet, hit "Upload" button
